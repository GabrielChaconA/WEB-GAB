SmartBudget-IA

Una plataforma de inteligencia financiera personal de próxima generación.
Nota importante: Esta es la versión 1.0 funcional sin IA, la integración de Inteligencia Artificial se usará luego, pero de momento es un sistema completamente operativo.

Project Purpose & Context

SmartBudget actúa como tu CFO (Director Financiero) personal, permitiéndote consolidar liquidez, fondos de emergencia, pagos recurrentes e inversiones diversificadas en un solo panel para conocer tu Patrimonio Neto (Net Worth) en tiempo real.

La aplicación ofrece una experiencia fluida (frictionless) al permitir el inicio de sesión seguro utilizando tu cuenta de Google, gracias a la implementación de autenticación delegada (OAuth 2.0).

Funcionalidad por Pestañas

Dashboard Principal (Home)

Panel central para ver tu panorama financiero global. Muestra tu Net Worth exacto y soporta conversión de divisas automática (USD/MXN).

Gestión de Fondos (Wallets/Funds)

Administra liquidez creando cajas personalizadas para organizar dinero, usando etiquetas de presupuesto y registrando flujos de caja manuales.

Panel de Inversiones

Rastrea criptomonedas, acciones, ETFs y apuestas deportivas. Calcula automáticamente el ROI/ROE basado en tu inversión inicial y valor actual.

Top de Inversiones (Gainers)

Se conecta con oráculos de precios para mostrar los activos con mejor rendimiento global para la toma de decisiones informadas.

Suscripciones (Memberships)

Lista todas las plataformas mensuales/anuales (Netflix, Spotify, etc.) para visualizar la fuga de capital y proyectar gastos fijos.

Análisis Avanzado (Analytics)

Gráficos dinámicos e históricos que comparan ingresos vs. gastos, mostrando tasas de ahorro y recomendaciones patrimoniales.

Desarrollo Individual

A diferencia de proyectos colaborativos, SmartBudget fue desarrollado de principio a fin de manera individual. Todo el ecosistema del sistema, incluyendo la arquitectura Backend y la API REST en Laravel, el diseño y programación del Frontend en Vue 3, así como la estructura y despliegue del proyecto, fueron realizados íntegramente por mí, sin la participación de otros integrantes de equipo.

Tecnologías Utilizadas

Vue.js 3 Laravel 11 TypeScript Tailwind CSS shadcn-vue PHP 8.2 MySQL 8.0 Docker ECharts

APIs de Terceros Integradas

  • Binance API & CoinGecko: Cotizaciones y gráficas de criptomonedas.
  • Finnhub: Telemetría del mercado bursátil tradicional (acciones, ETFs).
  • OddsAPI: Rastreo de momios en apuestas deportivas.
  • ExchangeRate-API: Conversión de divisas en tiempo real (USD/MXN) y coberturas.
  • Google OAuth 2.0: Autenticación delegada segura y rápida.

Infraestructura

Frontend (SPA): Vue 3 propulsado por Vite, usando vue-router y vue-i18n (Inglés/Español nativo).

Backend (API REST): Laravel Sanctum (tokens/cookies) y Laravel Socialite.

Despliegue: Entorno Dockerizado (Laravel Sail) e Infraestructura como Código (IaC) en render.yaml.
